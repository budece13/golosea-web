# golosea-web

Web de producción de **Golosea**, tienda de golosinas y chucherías en El Bonillo (Albacete).

Proyecto [Astro](https://astro.build). Incluye landing, chatbot (Claude), formulario de encargos vía WhatsApp, galería, mapa y páginas de blog y novedades gestionadas en Markdown.

## 🧞 Comandos

Todos se ejecutan desde la raíz del proyecto:

| Comando             | Acción                                           |
| :------------------ | :----------------------------------------------- |
| `npm install`       | Instala dependencias                             |
| `npm run dev`       | Servidor local en `localhost:4321`               |
| `npm run build`     | Compila el sitio de producción en `./dist/`      |
| `npm run preview`   | Previsualiza el build localmente                 |
